## 📞 Support & Troubleshooting

### Issue: Styles not showing?
**Solution**: Check tailwind.css is imported in main.jsx

### Issue: Mobile layout broken?
**Solution**: Verify responsive classes (sm:, md:, lg:) are applied

### Issue: Animations jerky?
**Solution**: Check browser hardware acceleration is enabled

### Issue: Build fails?
**Solution**: Run `npm install` and check Node version (16+)

For detailed help, see the documentation files in the root directory.
