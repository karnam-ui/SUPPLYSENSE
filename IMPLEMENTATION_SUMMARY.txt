================================================================================
                          QUICK START GUIDE
================================================================================

PREREQUISITES
  • Node.js 16+
  • npm 7+
  • Backend running on http://localhost:8000
  • Optional: Ollama for AI features

INSTALLATION
  1. cd frontend
  2. npm install

START DEVELOPMENT
  npm run dev
  → Dashboard at: http://localhost:5173

BUILD FOR PRODUCTION
  npm run build
  → Output: frontend/dist/
